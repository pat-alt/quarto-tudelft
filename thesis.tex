\subsection{Custom Callouts}\label{custom-callouts}

standard callout blocks like Note~\ref{nte-custom}.

\quartocalloutnte{nte-custom} 

\vspace{-3mm}\textbf{Note \ref*{nte-custom}: Current Limitations}\vspace{3mm}

This is a makeshift solution that relies on setting the default
behaviour of callout blocks to \texttt{callout-appearance:\ simple}. We
rely on the default style of callout blocks, e.g.~the
\texttt{.callout-tip} for questions, where the light bulb icon sort of
makes sense.

Ideally, we would like to have either custom callout blocks or custom
theorem environments. There is
\href{https://quarto.thecoatlessprofessor.com/custom-callout/qcustom-callout-faq.html}{coatless-quarto/custom-callouts},
which unfortunately does not support PDF, and
\href{https://github.com/ute/custom-numbered-blocks}{ute/custom-numbered-blocks},
which I could not get to work in combination with custom
cross-references. For custom cross-references, Quarto currently only
supports float environments, so no custom theorem environments are
supported yet.

\end{minipage}%
\end{tcolorbox}
